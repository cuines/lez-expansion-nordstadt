# LEZ Expansion Nordstadt - Project Charter

## Project Goal
Expand the city's Low Emission Zone (LEZ) to include the Nordstadt commercial and residential district, with the objective of reducing NO₂ levels in compliance with EU and German air quality regulations.

## Key Stakeholders
- Urban Planning Office
- Legal Department  
- Public Transport Authority
- Environmental Protection Agency
- City Council
- Local Residents and Businesses

## Regulatory Framework
This project is based on:
- EU Clean Air Directive 2008/50/EC
- German Federal Immission Control Act (BImSchG)
- 39th Ordinance on the Implementation of the Federal Immission Control Act (39. BImSchV) – Traffic prohibition ordinances
- Local clean air and action plans

## Project Scope
- Define new LEZ boundaries for Nordstadt district
- Draft legal ordinance text with implementation timeline
- Create updated LEZ boundary maps
- Analyze public transport impact and propose adjustments
- Develop public communication materials (FAQ)

## Repository Structure
This repository contains all project documentation, drafts, and final deliverables in a version-controlled, transparent manner.
